# Michael Tabano — Professional Portfolio

A monochrome portfolio site for Michael Tabano, built with Next.js, TypeScript, and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

The downloadable PDF is auto-generated from the same content as `/resume` via `npm run generate-resume`.

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```
